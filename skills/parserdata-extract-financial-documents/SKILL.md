  Extracts structured financial data from invoices, receipts, bank statements,
  3PL invoices, payout reports, and other financial PDFs or images using the

  JSON, Excel, CSV, or XML formats. Use when a user uploads or references
  financial documents and needs reliable field extraction, line items,
  validation, reconciliation, or reporting.
